
# Pickles

A browser-based pixel image editor implemented in [Calmm.js][calmm-js].

## Contents

  - [Getting Started](#getting-started)
  - [To do](#to-do)

## Getting started

```sh
git clone https://github.com/stuf/pickles.git
cd pickles
yarn
yarn start
```
